# Splunk SOC Lab

Hands-on Splunk SIEM project simulating enterprise SOC operations —
demonstrating real-world skills in alert triage, log analysis,
threat detection, and incident response.

## Sections

| # | Section | Description |
|---|---|---|
| 01 | SOC Dashboard | Splunk dashboards for endpoint & login monitoring |
| 02 | Log Analysis | Windows & Linux log analysis to detect anomalies |
| 03 | Threat Detection | MITRE ATT&CK mapping of all findings |
| 04 | Playbooks | Tier 1 SOC incident response playbooks |

## Tools Used
- Splunk Enterprise (Free Trial)
- Windows Event Logs / Linux Syslogs
- MITRE ATT&CK Framework v14
- SPL (Splunk Processing Language)

## Key Detections
- Brute force attacks from external IPs
- PowerShell download cradle execution
- Backdoor account creation
- Malicious cron reverse shell
- Privilege escalation via sudo
